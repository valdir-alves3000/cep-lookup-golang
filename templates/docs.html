Documentação da API CEP

Bem-vindo à API de Consulta de CEP!

Esta API permite consultar informações de endereços brasileiros através do CEP, integrando com BrasilAPI e ViaCEP.

Endpoints Disponíveis

GET /api/cep
Retorna informações detalhadas do endereço a partir do CEP fornecido.

Parâmetros da Query

cep: CEP a ser consultado (apenas números)
Exemplo: /api/cep?cep=01001000

Exemplo de Resposta

{
    "address": {
        "cep": "01001-000",
        "logradouro": "Praça da Sé",
        "bairro": "Sé",
        "localidade": "São Paulo",
        "uf": "SP"
    },
    "api": "brasilapi.com.br",
}

Códigos de Status

200 Sucesso - Retorna os dados do endereço
400 Erro - CEP não fornecido ou inválido
408 Erro - Timeout na consulta às APIs

Notas Importantes

  • O CEP deve ser fornecido apenas com números (8 dígitos)
  • A API possui timeout de 1 segundo
  • A consulta é feita simultaneamente em duas APIs (BrasilAPI e ViaCEP)
  • O resultado retornado será da API que responder primeiro

Exemplo de Uso

fetch('https://cep-lookup.vercel.app/api/cep?cep=01001000')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Erro:', error));
